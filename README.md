# docker-php7

A PHP7 docker file which installs useful command line tools, a number of php extensions and turns on apache SSL and mod rewrite

## Command line tools

- apt-utils 
- net-tools 
- vim 
- wget 
- curl

## Dependencies

- libfreetype6-dev 
- libjpeg62-turbo-dev 
- libmcrypt-dev 
- libpng12-dev 
- libxml2-dev 
- libcurl3 
- libcurl3-dev

## PHP Extensions

- pdo_mysql 
- gd 
- mcrypt 
- mysqli 
- zip 
- soap 
- calendar 
- curl 
- exif 
- gettext 
- sockets 
- opcache

## Apache Modules

- rewrite
- ssl
